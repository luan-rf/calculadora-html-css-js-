-- site que com html básico usando div, seções para mostrar uma calculadora, separando os 
botões de números, operadores, delete, erase com classes 

-- arquivo css com estilos personalizados semelhantes ao design aplicado para as calculadoras 
do iOS. 

-- arquivo JS com a lógica de programação para as interações de números/operadores/erase/delete 
indicação de buffer e atribuição de lógicas para as classes e funções utilizadas na calculadora. 



desenvolvido no VSCode, utilizando a extensão de preview para rodar no localHost enquanto 
o projeto foi sendo realizado.
